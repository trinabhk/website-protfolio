About My Work

I am a BCA student from Nepal working as a SEO Analyst at SmartSites Nepal. I bridge the gap between technical search engine optimization and front-end web updates. My daily work focuses on fixing the code and configuration issues that keep websites from ranking properly on Google.

On the front-end side, I do hands-on updates across platforms like WordPress, Shopify, Wix, Squarespace, and BigCommerce. I have basic experience with Shopify Liquid code and regularly manage critical SEO tags, canonical URLs, 'noindex' rules, and robots.txt file structures. I also run audits to track down and repair broken links (3xx redirects and 4xx errors) and optimize navigation menus to improve user experience.

Additionally, I handle the technical setup required to launch or verify websites. This includes working directly with DNS configurations and TXT records across hosting providers like GoDaddy, Cloudflare, WP Engine, and HostGator to verify properties smoothly in Google Search Console.

Core Technical Capabilities

Executing agile front-end optimization based on rigorous industry standard audits:

Sitemap & Link Management

Creating and organizing website sitemaps. For large online stores, I split sitemaps into neat groups (like separate maps for products, blogs, and regular pages) so search engines can easily find, read, and track every single page on the site.

Fixing Website & Indexing Errors

Fixing the technical issues sent over by project managers. I jump into the code to add or remove 'noindex' tags, fix broken redirect links, fix 'Soft 404' errors (like empty store categories that confuse Google), and make sure the site tells search engines exactly which links are the correct ones to show.

Google & Bing Dashboard Setup

Setting up new websites from scratch in webmaster tools. I handle the backend verification (proving to Google and Bing that we own the site), connect our sitemaps so search engines start reading the site immediately, and link everything up with Google Analytics to track traffic.
